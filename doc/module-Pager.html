Pager

分页控件 提供Ajax及本地数据分页功能
Source:

Example

<div class="pager-container"></div>
new Pager({
    main: lib.q('pager-container')[0],
    total: 10,
    onChange: function (e) {
        // load new date
        this.setPage(e.page);
        this.render();
    }
 }).render();
